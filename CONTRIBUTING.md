# Guia de Contribuição

Obrigado por querer contribuir com nosso projeto de Calculadora Simples!

## Como Começar

1. Faça um Fork do repositório
2. Clone seu fork: `git clone https://github.com/SEU-USER/aula-3-...git`
3. Crie um branch descritivo: `git switch -c feature/sua-feature`
4. Faça suas mudanças e commits semânticos
5. Push para seu fork: `git push origin feature/sua-feature`
6. Abra um Pull Request com descrição clara

## Padrões de Commit

- `feat:` nova funcionalidade
- `fix:` correção de bug
- `refactor:` refatoração de código
- `docs:` mudanças em documentação
- `test:` adição de testes

## Processo de Code Review

Todos os PRs precisam de review de pelo menos 1 membro do time antes do merge.
